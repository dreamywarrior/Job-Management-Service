# Job-Management-Service
Job Management microservice for an ML-powered Fake Job Posting Detection System built using FastAPI, SQLite, SQLAlchemy, and Jinja2, following Microservices and CQRS architecture.

## Setup & Installation

To run the complete application locally (including the Machine Learning engine), follow these steps:

1. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
2. **Train the Machine Learning Model:**
   The ML engine needs to be trained on the EMSCAD dataset before the prediction API can work.
   ```bash
   python ml/train_model.py
   ```
3. **Seed the Database (Optional but Recommended):**
   This script populates the database with a test user and actual fake job postings so you can immediately test the analytics dashboard.
   ```bash
   python seed_db.py
   ```
4. **Run the Server:**
   ```bash
   uvicorn app.main:app --reload
   ```
   Access the application at `http://127.0.0.1:8000`.

## Testing the Application (For Screenshots & Verification)

To test the system or take screenshots for the final report:

1. **Login:** Access the app and login as `test@example.com` (password: `testpassword` - if you ran the seed script).
2. **Command Side (Add Job):** Navigate to "Add New Job". You can input a fake job description (or use one already generated by the seed script).
3. **Query Side (Verify Job):** Navigate to "My Jobs", select a job, and click "Verify Job". The ML engine will analyze it and display a "Fake" or "Legitimate" tag alongside a Confidence Score.
4. **Analytics:** Check the "Dashboard" tab to view real-time Chart.js pie charts of the fraud detection history.
